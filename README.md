# CloudFlare Image Host - 现代化多用户图床系统

基于 Cloudflare Pages Functions + R2 + KV 构建的现代化图床系统，支持多用户、文件管理、分享链接等完整功能。

## ✨ 特性

- 🔐 **完整用户系统** - 邀请码注册、多级权限管理
- 📁 **强大文件管理** - 支持多种格式、批量操作、文件夹管理
- 🔗 **智能分享系统** - 密码保护、访问限制、有效期控制
- 🎨 **现代化界面** - 响应式设计、明暗主题、流畅动画
- ⚡ **高性能存储** - Cloudflare R2 + CDN 全球加速
- 📊 **可视化管理** - 实时统计、用户管理、系统监控
